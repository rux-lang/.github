# Rux

[![Ruxy](assets/ruxy.png)](https://github.com/rux-lang)

> **Fast. Compiled. Strongly Typed. Multi-Paradigm.**

Welcome to the official GitHub organization for **Rux**, a fast, compiled, strongly typed, multi-paradigm, general-purpose programming language.

---

## About Rux

Rux is designed to combine performance, safety, and flexibility in a modern development experience.

Whether you're building command-line tools, applications, libraries, services, or experimenting with language design, Rux provides the tools needed to write efficient and maintainable software.

### Core Characteristics

| Feature            | Description                                             |
| ------------------ | ------------------------------------------------------- |
| ⚡ Fast             | Native compiled performance                             |
| 🔒 Strongly Typed  | Type safety and improved reliability                    |
| 🏗️ Compiled       | Produces native executables                             |
| 🔄 Multi-Paradigm  | Procedural, functional, and object-oriented programming |
| 🌎 General-Purpose | Suitable for a wide range of applications               |

> [!NOTE]
> Rux is actively evolving and improving with every release.

---

## Philosophy

Rux is built around a simple set of principles:

* Performance matters.
* Developer experience matters.
* Readable code scales.
* Tooling should be first-class.
* Simplicity beats unnecessary complexity.
